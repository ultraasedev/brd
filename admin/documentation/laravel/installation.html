Installation

Installation Process

The following process will be used to install admin dashboard using package manager

1. Install Composer : Please install latest version of Composer from https://getcomposer.org/

2. Laravel : Create laravel project.

Note:- The Laravel project should be created in a localhost folder (such as htdocs, www, etc).

3. After, the Laravel Project is created. Extract the CORK ADMIN purchased file.

4. You will see the folder structure inside each demo. Kindly refer the following link file structure

5. Copy all the folders and files (such as public, resources, routes, stroage, webpack.mix.js). Replace them inside the project which we have created in point no. 2

6. After replacing all important files.

7. Follow these optional steps.

OPTION 1

a. Change Database name inside .env file.

b. Upload the SQL file to the Database.

Note:- Make sure to that you have alreardy created the database before uploading the SQL file.

OPTION 2

a. Access Command Prompt : Open command prompt.

b. Migrate Database : Run - php artisan migrate command to create login datatables.

c. Use Tinker to create login access.

Note:- The login access can be created as per your need.

8. Once you are logged in successfully. You can start further development.
