Getting Started

What is Starter Kit?

Starter kit are a set of pages, useful for developers to start development process from scratch. In Cork Admin Template each layout/demo includes base components which can be used easily according to your needs.

How does it work?

Open one of the starter pages, add necessary plugins. That's it. It's just faster and easier.

Cork Starter Kit includes 6 Layout : - Default, Alternate Menu, Boxed, Inverse, Breadcrumb, Blank Page.

  • Blank Page- This Layout is same as Default Layout listed above without any widgets.
  • Breadcrumb Layout - This layout contains HTML and CSS for breadcrumbs.
  • Boxed Layout - This is boxed version of default layout.
  • Alternate Menu - In this layout sidebar is either closed or open by default or it may contain a design modification.
